## Description

Tutorial to test the code.

## Installation

```bash
# paste this in your terminal
$ git clone git@github.com:RaulAquino207/java-topics-for-web.git
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
